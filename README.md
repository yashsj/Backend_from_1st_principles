# Backend from 1st Principles - Handwritten Notes

Custom handwritten notes based on **Srini's** legendary *Backend from First Principles* [YouTube playlist](https://youtube.com/playlist?list=PLui3EUkuMTPgZcV0QhQrOcwMPcBCcd_Q1&si=Ph6UG8tK98JwUtK5).

These notes have been requested multiple times by folks on X - so here they are!

---

## What Is This?

*Backend from First Principles* by Srini is one of the most respected backend engineering series on YouTube. It covers how backend systems are **actually built** - not just frameworks and tutorials, but the *why* behind every decision: networking, protocols, databases, concurrency, distributed systems, and more.

This repo contains **my personal, handwritten notes** made while going through the playlist. They're dense, opinionated, and include my own annotations and aha moments - not just copy-pasted theory.

**Work in Progress:** Starting with the first 10 parts to gauge interest and feedback. More coming soon!

---

## What's Covered (First 10 Parts)

The notes cover foundational backend concepts including:

- **HTTP Protocol** - How the web communicates
- **Routing** - Directing requests to the right handlers
- **Serialization & Deserialization** - JSON, data formats, and transformations
- **Authentication & Authorization** - Securing your APIs
- **Validation & Transformation** - Input handling and sanitization
- **Middlewares** - Request/response pipelines
- **Request Context** - Managing state across handlers
- **Handlers, Controllers & Services** - Code organization patterns
- **CRUD Operations** - Create, Read, Update, Delete deep dive
- **RESTful Architecture** - API design best practices

More topics from the full playlist - databases, caching, queuing, scaling, real-time systems, testing, DevOps - will be added based on response.
